# ➕ Programa de Suma en Python

Este es un programa básico desarrollado en Python que permite sumar dos números ingresados por el usuario desde el teclado.

---

## 📌 Descripción

El programa solicita al usuario dos números, realiza la operación de suma y muestra el resultado en pantalla.

Es ideal para personas que están comenzando a aprender programación en Python y desean entender conceptos básicos como:

- Variables
- Entrada de datos (`input`)
- Conversión de datos (`float`)
- Operaciones matemáticas
- Salida de datos (`print`)

---

## 🛠️ Requisitos

Para ejecutar este programa necesitas tener instalado:

- Python 3.x

Puedes descargar Python desde:

https://www.python.org/
